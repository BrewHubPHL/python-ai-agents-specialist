# Glossary

| Term | Definition |
|------|------------|
| **ADK** | Google Agent Development Kit — `LlmAgent`, workflows, tool binding |
| **FastMCP** | Python MCP server framework |
| **root_agent** | Exported `LlmAgent` discovered by `get_fast_api_app` |
| **Intent/Contract/Enforcement** | Three-layer agent architecture rubric |
| **bearer_client** | Supabase client scoped to member JWT |
| **HMAC bridge** | Signed requests between Next.js and Python |
| **ai_job_queue** | Postgres queue for async AI extraction jobs |
| **SKIP LOCKED** | Queue dequeue pattern for concurrent workers |
| **APP_COMMAND** | Coolify env to run alternate process from same image |
